## 🇧🇷 Português

O **DuoGestão** é uma aplicação web completa voltada para a organização pessoal, unindo um controle de despesas detalhado a uma lista de tarefas (To-Do List) dinâmica. O projeto foi construído com foco em arquitetura robusta, persistência de dados e uma experiência de usuário fluida.

### 🛠️ Arquitetura e Tecnologias
A aplicação foi desenvolvida utilizando **JavaScript Vanilla (ES6+)**, seguindo o padrão de projeto **MVC (Model-View-Controller)**. Isso garante uma separação clara entre a lógica de negócios, a interface do usuário e o controle de fluxo.

* **Model:** Gerencia o estado e a persistência via `localStorage`.
* **View:** Isola a manipulação do DOM e proteção contra XSS.
* **Controller:** Orquestra a interação entre o usuário e os dados.

### ✨ Principais Funcionalidades
* **Gestão de Despesas:** Persistência de dados, máscara de valor monetário (R$), estatísticas por categoria e filtros avançados (ano, mês, tipo).
* **Lista de Tarefas:** Edição inline, sistema de notificações (Toasts), filtros de status e animações suaves.
* **Design Responsivo:** Layout adaptável para dispositivos móveis e desktop com variáveis CSS para fácil manutenção.

## 🇺🇸 English

**DuoGestão** is a comprehensive web application for personal organization, combining detailed expense tracking with a dynamic To-Do List. The project focuses on robust architecture, data persistence, and a seamless user experience.

### 🛠️ Architecture & Technologies
The application was built using **Vanilla JavaScript (ES6+)**, following the **MVC (Model-View-Controller)** design pattern. This ensures a clean separation between business logic, UI, and flow control.

* **Model:** Handles state management and persistence via `localStorage`.
* **View:** Manages DOM manipulation and XSS protection.
* **Controller:** Orchestrates user interaction and data flow.

### ✨ Key Features
* **Expense Management:** Data persistence, currency input masking (BRL), category-based statistics, and advanced filtering (year, month, type).
* **To-Do List:** Inline editing, notification system (Toasts), status filters, and smooth animations.
* **Responsive Design:** Fully adaptable layout for mobile and desktop using CSS variables for easy theming.
